# booklist-app-javascript
Booklist app in pure javascript

<p> Basic app in pure javacript without using any framework. </p>

Stacks used:
  1) Javascript (ES6)
  2) Bootstrap
  3) fontawesome
  
Functionality:
  1) List
  2) Add
  3) Remove 
  4) Message display with timeout functionality (message removes after 3 sec)
